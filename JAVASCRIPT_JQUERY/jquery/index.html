Lấy nội dung

Cú pháp : var " tên biến " = $('class/id').text()/,html() ;

.text(); : Chỉ lấy nội dung chử k có định dạng

.html(); : Lấy nội dung chử kèm theo định dạng

Hàm này là hàm lấy nội dung và cũng là hám gán nội dung

Dùng hàm thay thế nội dung này

Cú pháp thay thế là: var title3 = $('.title3').html( "Nội Dung này Đã Bị Thay Thế.");

thêm bội dung vào trong thẻ html(noi dung) or text(noi dung)

Ví dụ -Bấm vào đây-

trong form, lấy dữ liệu k phải =.text() / .html() . Thay vào đó là: .val()

lấy dữ liệu trong biến value để hiển thị lên popup

Cú pháp của id và class ở trên ta có thể sủ dụng tương tự trên CSS (id cha id con thẻ p : #id #child p)
