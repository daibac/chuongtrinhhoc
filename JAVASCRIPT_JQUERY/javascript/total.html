.map()

Tạo ra một mảng mới bằng cách thực hiện một chức năng trên mỗi phần tử mảng.

Ví Dụ : mỗi phần tử đc nhân 2 rồi thêm vào màn hình 
		< p id="demo">
		< script>
		var numbers1 = [45, 4, 9, 16, 25];
		var numbers2 = numbers1.map(myFunction);

		document.getElementById("demo").innerHTML = numbers2;

		function myFunction(value) {
		  return value * 2;
		}< /script>

.filter(()

một mảng mới từ các phần tử có giá trị < /> / = .... số nào đó 

Ví Dụ : hiển thị các số lớn hơn 18
		< p id="demo">

		< script>
		var numbers = [45, 4, 9, 16, 25];
		var over18 = numbers.filter(myFunction);

		document.getElementById("demo").innerHTML = over18;

		function myFunction(value) {
		  return value > 18;
		}< /script>

.reduce ()

điều hành một chức năng trên mỗi phần tử mảng sản xuất (giảm nó để) một giá trị duy nhất.

Ví Dụ : 
		< p id="demo">
		< script>
		var numbers = [45, 4, 9, 16, 25];
		var sum = numbers.reduce(myFunction);

		document.getElementById("demo").innerHTML = "The sum is " + sum;

		function myFunction(total, value) {
		  return total + value;
		}< /script>

.every()

		var allOver18 = numbers.every(myFunction);

		document.getElementById("demo").innerHTML = "All over 18 is " + allOver18;

		function myFunction(value) {
		  return value > 18;
		}

Đối tượng toán học JavaScript

Math.round()

	Ví dụ : làm tròn lên 6 
	var x = 5.55556
	document.getElementById("demo").innerHTML = Math.round(x);

Math.pow()

	Ví dụ : Lũy thừ 3 của 6 
	document.getElementById("demo").innerHTML = Math.pơ(6,3); /// return 216

Math.sqrt(x)

	Ví dụ : trả về căn bậc hai của x: 
	document.getElementById("demo").innerHTML = Math.sqrt(64);      // returns 8

Math.abs(x)

	Ví dụ : trả về giá trị tuyệt đối (dương) của x: 
	document.getElementById("demo").innerHTML = Math.abs(-4.7);     // returns 4.7

Math.ceil(x)

	Ví dụ : trả về giá trị của x làm tròn  lên  đến số nguyên gần nhất của nó:
	document.getElementById("demo").innerHTML = Math.ceil(4.4);     // returns 5

Math.floor(x)

	Ví dụ : trả về giá trị của x được làm tròn  xuống  số nguyên gần nhất của nó:
	document.getElementById("demo").innerHTML = Math.ceil(4.4);     // returns 5

Math.sin(x)

	Ví dụ : trả về sin (giá trị từ -1 đến 1) của góc x (được tính bằng radian).
Nếu bạn muốn sử dụng độ thay vì radian, bạn phải chuyển đổi độ sang radian:
--Góc tính theo radian = Góc tính theo độ x PI / 180.
	document.getElementById("demo").innerHTML = Math.sin(90 * Math.PI / 180);     // returns 1 (the sine of 90 degrees)

Math.cos(x)

	Ví dụ : trả về cosin (giá trị từ -1 đến 1) của góc x (được tính bằng radian).
Nếu bạn muốn sử dụng độ thay vì radian, bạn phải chuyển đổi độ sang radian:
--Góc tính theo radian = Góc tính theo độ x PI / 180.
	document.getElementById("demo").innerHTML = Math.cos(0 * Math.PI / 180);     // returns 1 (the cos of 0 degrees)
